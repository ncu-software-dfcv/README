# NCU Software-DFCV Lab
### 南昌大学软件学院｜数字取证与计算机视觉实验室
**School of Software, Nanchang University • Digital Forensics & Computer Vision Lab**

---

## 🔬 实验室简介
本实验室聚焦**数字取证、多媒体安全、AIGC 内容溯源、深度伪造检测、分布式计算与计算机视觉**等方向，依托**数据安全技术江西省重点实验室**与南昌大学“双一流”科研平台，开展前沿理论创新与工程应用研究。团队由多名高水平教师、硕博研究生组成，科研体系完善、算力资源充足、成果突出，在国内外多媒体安全与 AI 安全领域具有广泛学术影响力。

---

## 🎯 研究方向
- AIGC 生成机理与伪造内容取证
- Deepfake 检测、语义解耦与泛化性提升
- 数字图像/视频取证与反取证
- 多媒体内容真实性与完整性验证
- AI 安全、对抗鲁棒性与隐私保护
- 分布式计算、多模态语义分析与智能系统

## 🖥️ 科研平台与基础设施
- **GPU 服务器集群**：8 台高性能 GPU 计算节点 + 多台通用 Linux 服务器
- **数据资源**：数十万级图像/视频专题数据库，覆盖主流伪造图像与多模态数据集
- 依托平台：数据安全技术江西省重点实验室、CERNET 江西主节点网络环境
- 支撑能力：大规模深度学习训练、消融实验、跨模态语义分析、系统研发与验证
